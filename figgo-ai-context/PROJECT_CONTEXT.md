# CONTEXTO GERAL

Nome do app: Figgo

Tema:
Álbum da Copa do Mundo 2026.

Problema resolvido:
Colecionadores não têm um sistema moderno para:
- controlar álbum
- marcar repetidas
- achar trocas
- descobrir pontos de troca
- interagir com comunidade

Solução:
Aplicativo completo com gamificação e visual premium.
